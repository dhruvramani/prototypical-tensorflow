# Prototypical-Tensorflow - for NLP

Prototypical networks for few shot learning currently with a single LSTM encoder and trained on `Reuters` dataset.

File : `main_nlp.py`

## Installation
```shell
pip3 install tensorflow
pip3 install keras

```

## Running
```shell
python3 main_nlp.py
```
